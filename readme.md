## 1. 프로젝트 소개
- 기존 여행사 사이트인 '마이리얼트립'을 '경주여행'이라는 테마로 리뉴얼하는 프로젝트입니다.
- 2030 세대를 대상으로 대한민국의 대표 여행지인 경주를 매력적으로 홍보하기 위해 이 프로젝트를 기획하게 되었습니다.
- 리뉴얼된 사이트는 젊은 세대가 선호할 만한 콘텐츠와 디자인을 통해 흥미를 끌고, 경주 여행에 대한 관심을 높이는 것을 목표로 합니다.
